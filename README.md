## 功能模块

### 首页数据展示

- 用户创建时间
- 用户信息
- 数据信息
- 爬取数据时间的可视化展示

### 可视化分析

- **薪资情况：**
  - 支持选择不同学历查看岗位薪资分布
  - 展示形式：柱状图、饼图

- **企业情况：**
  - 根据职位查看主要招聘城市
  - 展示公司规模和职位在不同城市比例
  - 展示形式：饼图

- **学历情况：**
  - 查看各职位对学历和工作经验的要求
  - 展示形式：漏斗图、矩形树图
